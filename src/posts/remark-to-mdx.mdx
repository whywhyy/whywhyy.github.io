## MDX?
Syntax = Markdown JSX 


## 왜 MDX?
gatsby 에서는 포스트 작성시 md 보다는 mdx 를 사용하면 더 좋다(?). mdx 에서는 컴포넌트 사용이 가능하기 때문이며, 기존 remark plugin들도 사용할수 있으니 그냥 mdx 쓰도록 하자.
[사용방법 (참고1)](https://mdxjs.com/getting-started#hello-world)
[(참고2)](https://www.gatsbyjs.org/docs/mdx/)

## 컨버팅하자

### 어떻게??
기존 markdown을 mdx 로 바꾸자!

`allMarkdownRemark -> allMdx`

`markdownRemark ->mdx `

```js
// blog-post-template.js
import { MDXRenderer } from "gatsby-plugin-mdx" //highlight-line

html -> body 

<section dangerouslySetInnerHTML={{ __html: post.html }} />
-> <MDXRenderer>{post.body}</MDXRenderer> 
```

### install & uninstall

```bash
npm install --save gatsby-plugin-mdx gatsby-plugin-feed-mdx @mdx-js/mdx @mdx-js/react
```
```bash
npm uninstall --save gatsby-transformer-remark gatsby-plugin-feed
```
