## Code Jam

### 테스팅코드를 통해 배우다.

 TDD 기반으로 된 문제를 푸는 정말 좋은 기회였다.

 문제를 풀기위해 테스트 코드를 읽어야 하는데, 이 테스팅 코드가 작성되어 동작하는것을 보는것 자체가 일단은 배울점이다.

 클래스를 커스터마이징 하기위해 repr, len, lt 를 다루는 점도 좋았다.

 클래스 객체를 다루는 set, get 도 다뤄보는 좋은 기회였다.

 문제중 class 의 내부 id 를 핸들링하는것도 좋았다.
